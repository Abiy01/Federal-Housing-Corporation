## Quick reference (form field names)

When typing into the admin form, map as follows:

| Draft label | Form / API field |
|-------------|------------------|
| price (ETB) | `price` (number only, no commas) |
| availability | `availability` — exactly `rent` or `buy` |
| type | `type` — exactly `House`, `Apartment`, `Office`, or `Villa` |
| amenities | Select or add strings that **exactly** match [`admin/src/constants/amenities.js`](admin/src/constants/amenities.js) |

Replace placeholder phones with your official contact policy before publishing if required.
